# Инструкция к тестированию

Для успешного тестирования необходимо:

1. Создать пользователей с именами **Менеджер1** (с добавленной ролью ВКМ_ОбслуживаниеКлиентов), **ИвановИ** (с добавленной ролью ВКМ_ОбслуживаниеКлиентов), **Бухгалтер** (с добавленной ролью ВКМ_МассовоеСозданиеРеализаций).
2. Заполнить справочник Сотрудники

| Код | Наименование | Должность |
| --- | ------------ | --------- |
| 000000003 | Специалист 1 | Специалист |
| 000000004 | Специалист 2 | Специалист |

При необходимости заполнить справочник Должности

3. Заполнить справочник Контрагенты

| Код | Наименование |
| --- | ------------ |
| 000000001 | контрагент1 |
| 000000002 | контрагент2 |
| 000000003 | Контрагент 3 |

4. Заполнить справочник Договоры контрагентов

| Код | Наименование | Контрагент | Организация | Вид договора | Дата начала действия | Дата начала действия | Сумма абонентской платы  | Стоимость часа работы |
| --- | ------------ | ---------- | ----------- | ------------ | -------------------- | -------------------- | -----------------------  | ---------------------- |
| 000000003 | 3 (С АО срок 2026 год) | контрагент1 | организация1 | Абонентское обслуживание | 01.01.2026 | 31.12.2026 | 100,00 | 250,00 |
| 000000004 | 4 (С АО срок 2026 год) | контрагент2 | организация1 | Абонентское обслуживание | 01.01.2026 | 31.12.2026 | 1 000,00 | 250,00 |
| 000000005 | 5(с АО срок 2026) | Контрагент 3 | организация1 | Абонентское обслуживание | 01.01.2026 | 31.12.2026 | 10 000,00 | 100,00 |

5. Запуск теста можно производить по отдельности, или через АвтотестОбщий.feature, добавив кастомные шаги в библиотеку шагов.
